O que são Media queries?

As media queries permitem que o dev aplique estilos diferentes dependendo das caracteristicas do dispositivo ou tela. O caso mais comum é alterar o layout conforme a largura da tela, para que o site fique responsivo e adaptado a celulares, tablets e monitores.

  • @media(max-width: 768px): Aplica estilos para telas pequenas (smartphones e tablets em modo retrato.)
  • @media (min-width: 1200px): Aplica estilos para telas grandes (monitores Full HD, 2k e4K.)
  • Tambem é possivel usar condições como orientation: landscape ou prefers-color-schema.
