CSS Tranform e Translate

O que é translate?

A função translate no CSS é usada para mover um elemento no espaço bidimensional (horizontalmente e verticalmente). é especificada em unidades px, % ou em .

Tipos de translate:

  • translateX(valor): Move o elemento ao longodo eixo X.
  • translateY(valor): |Move o elemento ao logo do eixo Y.
  • translate(x,y): Movo o elemento ao longo do eixo X e Y.
