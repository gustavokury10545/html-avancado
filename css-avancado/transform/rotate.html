CSS Transform e Rotate

A propriedade transform no CSS permite aplicar transformações a um elemento, como redicionamento, rotação, inclinação ou movimentação. É uma ferramenta poderosa que permite criar efeitos visuais sem a necessidade de scripts.

Oque é rotate?

A funçãoRotate dentro da propriedade tranform é usada para girar um elemento em torno de um ponto de origem. A rotação é especificada em graus deg, mas também pode usar outras unidades como radianos rad ou frações de volta turn.

Exemplos básicos

transform: rotate(45deg) - Este codigo gira o elemento em 45 graus no sentido horário.
