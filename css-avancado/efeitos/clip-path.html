Clip-path

O clip-path é uma propriedade CSS que permite criar formas complexas cortando um elemento de acordo com um caminho específico

Como funciona o clip-path

O clip-path define uma area visual de um elemento, escondendo o que esta fora dessa área. Existem varias formas que podemos usar com clip-path, como:

  • polygon(): Define uma forma usando uma lista de cordenadas que formam um polígono.
  • circle(): Define uma forma circular.
  • ellipse(): Define uma forma eliptica.
  • inset(): Define um retângulo interno com uma margem opcional.
